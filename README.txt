Product compare for woocommerce.

== Description ==

Now, you can easily compare multiple product in woocommerce shop. Just add active the plugin then put shortcode in a page. 

Now, customer will see a compare button with all product, and will see the compare page with all other products.

## Features: ##

**Shortcode** Paste [pcwc_comparison/] shortcode to a page and see the product compare table.

== Installation ==

= Modern Way: =
1. Go to the WordPress Dashboard "Add New Plugin" section.
2. Search For "AsCode Woo Calculator". 
3. Install, then Activate it.

= Old Way: =
1. Upload `AsCode Woo Calculator` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
